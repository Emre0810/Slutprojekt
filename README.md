# Matte 1c Hemsida

## Beskrivning

Detta är en hemsida om Matematik 1c. Den innehåller förklaringar och funktioner och ett test där användaren kan träna sina kunskaper.

## Funktioner

- Navigationsmeny
- Genomgångar av matematik
- Quiz med poängsystem (JavaScript)
- Formulär med XSS-skydd
- Responsiv design

- JavaScript
